# 过拟合与正则化

## 章节目标
理解过拟合产生的原因，掌握 L1 正则化、L2 正则化和模型复杂度控制的基本思想。

## 核心概念
过拟合指模型在训练集上表现很好，但在新数据上表现较差。正则化通过限制参数规模或模型复杂度，提高泛化能力。

## 关键知识点
- 过拟合：模型记住训练数据噪声，而不是学习一般规律。
- L1 正则化：倾向于产生稀疏参数。
- L2 正则化：倾向于让参数更小更平滑。

## 常见易错点
- 训练误差越低不一定代表模型越好。
- 正则化强度过大可能导致欠拟合。

## 基础练习
1. 过拟合和欠拟合有什么区别？
2. L1 正则化和 L2 正则化有什么不同？

## 推荐实践任务
训练一个逻辑回归模型，比较不同正则化强度下的训练准确率和测试准确率。
