# 线性回归

## 章节目标
理解线性回归如何用线性模型拟合连续数值目标，掌握最小二乘法、均方误差和梯度下降的基本思想。

## 核心概念
线性回归用于预测连续值，例如房价、温度或销售额。它假设输出可以由输入特征的线性组合近似表示。

## 线性模型
线性模型把多个输入特征按权重加权求和，再加上偏置项。单变量形式可以写成 y = wx + b，多变量形式可以写成 y = w1x1 + w2x2 + ... + b。权重表示每个特征对预测值的影响方向和强度。

## 最小二乘法
最小二乘法希望找到一组参数，使预测值和真实值之间的平方误差总和最小。平方误差会放大较大的错误，因此模型会倾向于减少明显偏离的预测。

## 梯度下降
当直接求解析解不方便时，可以使用梯度下降。梯度表示损失函数上升最快的方向，参数更新时沿负梯度方向移动，从而逐步降低损失。

## 线性回归代码实现
Python 实现线性回归时，可以先准备特征矩阵 X 和标签 y，再定义预测函数、均方误差损失和参数更新循环。初学阶段可先用 NumPy 手写一轮梯度下降，再对比 scikit-learn 的 LinearRegression。

## 关键知识点
- 线性模型：用特征的线性组合预测连续值。
- 最小二乘法：最小化预测误差平方和。
- 梯度下降：通过迭代更新参数降低损失。
- 线性回归代码实现：用 NumPy 或 scikit-learn 完成训练与预测。

## 常见易错点
- 把线性回归用于类别标签预测。
- 只关注拟合训练集，忽略测试集误差。
- 学习率过大导致损失震荡，学习率过小导致训练很慢。

## 基础练习

2. 为什么均方误差适合衡量连续值预测误差？
3. 梯度下降中学习率过大会发生什么？
