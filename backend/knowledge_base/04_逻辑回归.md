# 逻辑回归

## 章节目标
理解逻辑回归如何解决二分类问题，掌握 Sigmoid 函数、交叉熵损失、梯度下降优化和 Python 实现思路。

## 核心概念
逻辑回归虽然名字中有“回归”，但通常用于分类。它先计算特征的线性组合，再通过 Sigmoid 函数映射为 0 到 1 之间的概率，最后根据阈值输出类别。

## 分类问题与逻辑回归思想
分类问题的目标是把样本分到离散类别中。对于二分类任务，模型需要判断样本属于正类还是负类。逻辑回归的核心思想是：先用线性模型得到分数 z，再把 z 转换成正类概率 p。

## Sigmoid 函数
Sigmoid 函数的形式是 sigmoid(z) = 1 / (1 + e^(-z))。当 z 很大时，输出接近 1；当 z 很小时，输出接近 0；当 z 等于 0 时，输出为 0.5。它用于解释逻辑回归如何将线性输出映射为概率。

## 逻辑回归损失函数
逻辑回归常用交叉熵损失函数。若真实标签为 1，希望预测概率接近 1；若真实标签为 0，希望预测概率接近 0。交叉熵会对自信但错误的预测给出较大惩罚。

## 梯度下降优化
逻辑回归通过梯度下降更新权重和偏置。每轮训练先计算预测概率，再计算损失关于参数的梯度，最后沿负梯度方向更新参数。学习率控制每次更新的步长。

## Python 代码实现思路
实现逻辑回归可以分为五步：准备特征矩阵和标签，定义 Sigmoid 函数，计算预测概率，计算交叉熵损失，使用梯度下降更新参数。初学者可以先用 NumPy 手写，再使用 scikit-learn 的 LogisticRegression 做对比。

## 关键知识点
- 分类问题与逻辑回归思想：用概率描述样本属于正类的可能性。
- Sigmoid 函数：把线性输出压缩到 0 到 1。
- 逻辑回归损失函数：用交叉熵衡量概率预测错误。
- 梯度下降优化：迭代更新参数，降低损失。
- 逻辑回归代码实现：用 NumPy 或 scikit-learn 完成训练。

## 常见易错点
- 误以为逻辑回归只能做数值预测。
- 忘记 Sigmoid 输出是概率，不是最终类别。
- 把均方误差直接套到逻辑回归分类任务上。
- 学习率设置不合理，导致训练不稳定。
