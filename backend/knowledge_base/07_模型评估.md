# 模型评估

## 章节目标
掌握分类模型常用评估指标，理解准确率、精确率、召回率和交叉验证的适用场景。

## 核心概念
模型评估用于判断模型在未知数据上的表现。单一指标可能无法完整反映模型质量，需要结合任务目标选择合适指标。

## 关键知识点
- 准确率：预测正确样本占总样本的比例。
- 精确率与召回率：分别关注预测为正的可靠性和正类样本被找回的比例。
- 交叉验证：多次划分训练和验证数据，获得更稳定的评估。

## 常见易错点
- 在类别极不平衡时只看准确率。
- 把验证集和测试集职责混淆。

## 基础练习
1. 类别不平衡时为什么准确率可能误导？
2. 召回率高代表什么？

## 推荐实践任务
用 scikit-learn 计算混淆矩阵、精确率、召回率和 F1 值。
